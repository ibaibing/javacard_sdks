The information in this README pertains to the Binary Release and the Binary Release with Cryptography Extensions of the Java Card 2.2 Development Kit.

  1. Supported Platforms

    • Windows® NT 4.0 with Service Pack 6
    • Solaris® 8 (SunOS 5.8) Operating Environment (SPARCTM platforms)

    Java Development Kit: Java2 SDK, Standard Edition (version 1.3)

    NOTE: You can download the supported JDK from http://java.sun.com/j2se/.

    [Top]

  2. Installation Instructions and Setting Up Your Environment

    3. For instructions on installing the binary release, see Chapter 2, "Installation" in the User's Guide that is included with the Java Card 2.2 Development Kit. This chapter also describes any additional tools you might need for setting up your environment.

    [Top]

  3. Product Overview

    4. The Java Card 2.2 Development Kit is a suite of tools for developing applets based on the Java Card 2.2 Application Programming Interface.

    Please refer to the User's Guide that is included with the Java Card 2.2 Development Kit for more information on using these tools.

    [Top]

  4. Where to Find Product Documentation

    5. In the Java Card 2.2 Development Kit package, information components may be in HTML, PDF, or text format:

    Text files can be viewed in your web browser or any simple text editor.

    HTML files also can be viewed in your web browser.

    PDF files can be viewed in your web browser with an appropriate plugin or in Adobe Acrobat Reader. Most recent browsers include the PDF reader plugin. If your browser does not, you can download the plugin from the browser vendor's website or the Adobe website at:

    http://www.adobe.com/products/acrobat/readstep.html

    Documentation files packaged in the product were current when the product shipped. See the Java Card website for any updates.

    • Product Readme (this document)

      If you installed the Java Card 2.2 Development Kit, the Readme is located in:

      <java_card_kit-2_2>\README.html (Windows example)
      <java_card_kit-2_2>/README.html (Solaris example)

      where:

      <java_card_kit-2_2> is the directory containing the installed Java Card 2.2 Development Kit product.

    • Release Notes

      Contains important information on code and documentation changes, known issues, and technical notes.

      If you installed the Java Card 2.2 Development Kit, the Release Notes are located in:

      <java_card_kit-2_2>\RELEASENOTES.html (Windows example)

      <java_card_kit-2_2>/RELEASENOTES.html (Solaris example)

    • Product Documentation
       
      • Java CardTM 2.2 Development Kit User's Guide
      • Java CardTM 2.2 Application Programming Notes

        Located in:
        <java_card_kit-2_2>\doc\en\guides\ (Windows example)
        <java_card_kit-2_2>/doc/en/guides/ (Solaris example)

      • Java CardTM 2.2 Off-Card Verifier White Paper
      • Java CardTM 2.2 RMI Client Application Programming Interface

        Located in:
        <java_card_kit-2_2>\doc\en\whitepapers\ (Windows example)
        <java_card_kit-2_2>/doc/en/whitepapers/ (Solaris example)

    • The Technology FAQ is available at:

      http://java.sun.com/products/javacard/faq.html

  6. How to Send Feedback and Questions

    7. To report a new bug or submit a feature request:

    • Simply fill out the details in the easy-to-use form on Sun's website at:
      http://java.sun.com/cgi-bin/bugreport.cgi

     

    To get an answer to a question which is not about a bug or a feature request:

    • We recommend you first check the Release Notes, then the Frequently Asked Questions (the Technology FAQ). This is often the fastest way to locate an answer. The Release Notes are included in the download bundle.
